Water Prediction System

The main task of the project is to build a robust application (a machine learning model) which could predict water consumption of the upcoming days (future days) using parameters like historic data of water consumption, changes in weather and population etc . The project is built using machine learning models. The project is implemented with Ensemble model which predicts accurate results. This project could act as an better extension to the current traditional water management system to manage and distribute water to a particular region and solves water scarcity and correct distribution of water.

  • Water consumption is increasing due to population growth and industrialisation.
  • Excessive demand is putting the present infrastructure under pressure.
  • Emerging Machine Learning Algorithms can be used to better model real life problems .
  • Core idea is to create a model using machine learning techniques that can accurately predict long term water usage.
  • Scope of project is to help to efficiently use the existing infrastructure to supply required amount of water,using the predicted data as a reference.
  • Project is primarily targeted to be used as a tool to assist authorities to make better decisions to distribute water.
  • Key idea to identify potential leakages and areas where water stress is the highest.
